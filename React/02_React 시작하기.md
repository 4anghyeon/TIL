# React 시작하기
## 필요 항목
### Node.js
- Webpack과 Bable과 같은 도구들이 자바스크립트 런타임 (서버사이드)기반으로 만들어져 있어서 Node.js를 설치해야함.
### Yarn
- 패키지 매니저 도구인 npm의 개선된 버전(?)
  - Yarn 설치 방법
    1. cmd > npm install -g yarn
    2. yarn --version 으로 설치되었는지 확인
### create-react-app
- react 앱을 만들어 주는 도구
- npm install -g create-react-app
- or
- yarn global add create-react-app
    - 여기서 -g 혹은 global은 package를 전역적으로 설치한다. 특정 디렉토리에서만 설치하고 싶다면 local로 설치하면 된다.


## React App 만들고 실행하기
1. 원하는 폴더에서 CMD 창을 실행시킨다.
2. create-react-app 프로젝트 명
3. cd 프로젝트 명
4. yarn start
